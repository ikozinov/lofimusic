# REQ-001: Портирование Lofimusic на Flutter

**Type**: requirement  
**Status**: in_progress  

## Description
Разработка кроссплатформенного клона веб-приложения `lofimusic.app` (оригинал написан на Go с использованием `go-app` PWA) полностью на фреймворке **Flutter**.
Целевые платформы: **macOS (основная приоритетная десктоп-версия)**, Web, Windows, Linux, iOS и Android.

---

## 1. Функциональные требования

### 1.1. Интеграция с YouTube (Стриминг)
- Воспроизведение живых трансляций (YouTube Live Streams) радиостанций по их идентификаторам из оригинальной базы данных.
- Фоновый запуск воспроизведения (автозапуск при выборе станции, если разрешено платформой).
- Фоновый оверлей: видеопоток должен растягиваться на весь задний план приложения с эффектом затухания по краям (сверху и снизу) с помощью градиента.

### 1.2. Панель навигации (Sidebar)
- Фиксированная панель слева шириной `282px` на экранах десктопа/планшета.
- На мобильных экранах (ширина `< 500px`) трансформируется в скрываемое боковое меню (Drawer / Hamburger menu).
- Содержимое сайдбара:
  - Логотип-заголовок **Lofimusic** (ссылка на главную страницу, которая сбрасывает/шафлит станции).
  - Вертикальный список доступных радиостанций (с иконкой Play и подсветкой активной станции).
  - Ссылки поддержки внизу панели:
    - *Buy me a coffee*
    - *Donate cryptos*
    - *GitHub*
    - *Twitter*

### 1.3. Панель управления плеером (Controls)
- **Кнопка Back**: Возврат к предыдущей воспроизводимой станции (через историю навигации). Скрывается/отключается, если истории переходов нет.
- **Кнопка Play/Pause**: Запуск и приостановка видео/аудио потока.
- **Центральная кнопка Shuffle**: Выбор и переход на случайную станцию.
- **Регулятор громкости**:
  - Иконка динамика (меняет вид в зависимости от уровня: Выкл / Тихо / Средне / Громко).
  - Клик по иконке переключает режим Mute / Unmute.
  - Ползунок (Slider) регулировки громкости (0-100%).
- На мобильных экранах кнопки Back и ползунок громкости скрываются (как в веб-оригинале).

### 1.4. Информационные карточки (Lore Cards)
- В левом верхнем углу экрана отображается оверлей с описанием текущей станции/художника.
- Карточки сменяются циклически: отображаются в течение `10 секунд`, исчезают на `5 секунд` (плавная анимация opacity).
- В правом нижнем углу отображается большой заголовок с названием радиостанции и иконками соцсетей (YouTube, Spotify, Discord, Instagram и др.).

### 1.5. Хранение состояния (Persistence)
- Сохранение уровня громкости пользователя.
- Сохранение последней воспроизводимой радиостанции для восстановления при следующем запуске.

## 2. Дизайн и UI/UX (Aesthetics)
- **Тема**: Глубокий черный фон (`#000000`) с контрастным белым/светло-серым текстом.
- **Шрифт**: Montserrat (через `google_fonts`).
- **Свечение (Glow)**: Текстовые тени (`text-shadow`) для эффекта неонового свечения заголовков на фоне темного видео.
- **Анимации**:
  - Плавные наведения (hover) на элементы управления и ссылки с эффектом масштабирования (`scale`).
  - Анимация пульсации/прыжка (bounce) для кнопки обновления/статуса.
  - Плавное появление (Fade-in/out) информационных карточек.

## 3. Архитектура и стек технологий (Flutter)

### 3.1. Зависимости (`pubspec.yaml`)
- `google_fonts`: Для подключения Montserrat.
- `shared_preferences`: Для локального хранения настроек (громкость, последняя станция).
- `youtube_player_iframe` / `flutter_inappwebview`:
  - `youtube_player_iframe` — для Web, iOS, Android.
  - `flutter_inappwebview` (или специализированный десктопный WebView) — для качественного воспроизведения YouTube Iframe API на macOS, Windows, Linux.
- `url_launcher`: Для перехода по социальным ссылкам и ссылкам донатов.
- `provider` или `riverpod`: Для реактивного управления состоянием плеера.

### 3.2. Структура директорий
```text
lib/
├── main.dart                 # Инициализация приложения и конфигурация роутинга
├── models/
│   └── radio_station.dart    # Модель радиостанции и социалок (конвертация live.go)
├── providers/
│   └── player_provider.dart  # Бизнес-логика плеера, управление громкостью и историей
├── screens/
│   └── home_screen.dart      # Главный экран (Shell c плеером и боковой панелью)
├── widgets/
│   ├── youtube_player_bg.dart# Виджет заднего фона с YouTube WebView
│   ├── sidebar_navigation.dart# Боковая панель выбора радио
│   ├── player_controls.dart  # Кнопки Play/Pause, Shuffle, Volume Slider
│   ├── info_overlay.dart     # Сменяющиеся лор-карточки (вверху слева)
│   └── title_overlay.dart    # Заголовок станции и ссылки (внизу справа)
```
